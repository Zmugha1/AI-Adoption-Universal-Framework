# VTCO Methodology Guide

## What is VTCO?

**V**erb → **T**ask → **C**onstraint → **O**utcome

A structured approach to defining and executing ML/data work for DR Data Decision Intelligence.

## Before Every Task

Define:

```
VERB:       [Create | Update | Delete | Analyze | Validate | Deploy]
TASK:       [Specific description of what you're doing]
CONSTRAINT: [Limitations, requirements, boundaries]
OUTCOME:    [Expected result/deliverable]
```

## MCP Tools

- `define_vtco` - Define a VTCO task
- `assess_risk` - Assess risk before proceeding
- `validate_artifact` - Validate outputs against schema
