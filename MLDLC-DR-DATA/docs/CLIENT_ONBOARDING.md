# Client Onboarding - DR Data Decision Intelligence

## Legal Compliance

Every decision MUST be:
- Auditable
- Traceable
- Transparent

No blackbox operations allowed.

## Framework Components

1. **VTCO** - Structured task definition
2. **Risk Matrix** - RED/YELLOW/GREEN classification
3. **Schema Validation** - JSON schema enforcement
4. **Lineage** - Data transformation tracking
5. **Audit** - Complete audit trail in data/audit/
